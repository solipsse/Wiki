# standard library
!!! tip
    [C header file explained: CS50][manual]{target="_blank"}

## `<stdlib.h>`
Memory allocation functions like `malloc()`, `calloc()`.
### escape sequence
```
\n: Newline
\t: Horizontal tab
\": Double quote
\\: Backslash
\a: Alert (bell)
\b: Backspace
\f: Form feed
\r: Carriage return
\v: Vertical tab
```
## `<string.h>`
 String manipulation functions like `strcpy()`, `strlen()`.
## `<math.h>`
 Mathematical functions like `sqrt()`, `pow()`.
